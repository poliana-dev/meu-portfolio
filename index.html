API para Gestão de Personagens, Filmes, Cenários, Músicas e mais. Labs

Esta API foi desenvolvida para gerenciar informações sobre filmes, personagens, acessórios, cenários, músicas e vestimentas/figurinos. Através do backend, é possível cadastrar, listar e apagar itens em cada módulo, oferecendo flexibilidade na administração dos dados. No frontend, a API permite apenas a visualização dessas informações com a estrutura RESTful para facilitar a integração e o acesso eficiente aos dados.
